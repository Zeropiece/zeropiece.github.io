CS144 Honor Code Discussion (April 3, 2023)

Q: Are you allowed to look at other peoples code? Or, if youre helping someone, can you look at their code?

A: Please adhere to the rules given in the lab assignment, which is: under no circumstances look at OR show OR post your code to other students in the class, OR look at code posted by students in prior quarters of the class (on the Internet or elsewhere).

Exception: Writing pseudocode longhand on a whiteboard or sheet of paper is OK BUT MUST BE DISCLOSED in the lab writeup.

Q: Can we share test cases and output of our system given various inputs?

A: Totally fine. There will be extra credit on some lab assignments for contributing new test cases that catch real bugs (not already caught by the testsuite).

BUT: Please dont share an actual testing script (which is code) except by submitting it to the course staff, who MAY choose to publish it to everybody.

Q: How nice can we be on Ed?

A: Anything overt does not greatly worry me. If you go over the line with niceness, well tell you. In helping people, please do consider what would be most helpful to their LEARNING (vs. just giving them the answer, which is not as helpful).

Q: What kinds of conversations about general approaches, principles, etc., are permissible?

A: Please disclose any substantive help that any other student gave you in your writeup. But broad conversations about different approaches and their pros and cons are usually very helpful for learning (and encouraged)! But make sure to credit anybody who helps you substantially.

Q: How about getting documentation online about APIs, e.g. a salt function so I googled the parameters?

A: If you find substantial help from an online reference or tutorial, please make sure to credit it in your writeup. Credit it SPECIFICALLY: I got help from GitHub is not acceptable (especially if that means I found the solutions from a prior year on GitHub and Im not telling you where).

Q: How about using online APIs, such as ChatGPT, Bard, Claude, GitHub Copilot, etc.?

A: This is equivalent to consulting a student who took this class in a prior quarter (and has solved the assignments). Any substantial help must be credited by name (and include the prompt given to ChatGPT etc.), and please do not ask for code that solves any of the assignments.
